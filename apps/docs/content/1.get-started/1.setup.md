## Enable the WebSocket Server

::navigate
**In Streamer.bot**, navigate to `Servers/Clients > WebSocket Server`
::

Ensure your WebSocket Server is enabled and running, or click `Start Server`

::warning{to=#initialize-a-client}
If you changed the `Host`, `Port` or `Endpoint` settings, make sure to follow the [Custom Settings](#initialize-a-client) guide below!
::

## Initialize a Client

::read-more{to=/api/config}
Explore all client [Configuration Settings](/api/config) on the API reference page
::

::tabs
:::tabs-item{label=Default icon=mdi:rocket}

You can initialize a new client instance with the `StreamerbotClient()`{lang=ts} constructor:

```ts [example.js]
const client = new StreamerbotClient();
```

::::success
The client will **automatically** connect to the default WebSocket server location at `ws://127.0.0.1:8080/`
::::
:::

:::tabs-item{label="Custom Settings" icon=mdi:cogs}

Initialize a new client with the `StreamerbotClient()`{lang=ts} constructor, passing in your custom configuration settings as the first argument:

```ts [example.js]
const client = new StreamerbotClient({
  host: '127.0.0.1',
  port: 9001,
  endpoint: '/',
});
```

::::success
The client will **automatically** connect to the configured location
::::
:::
