## Subscription Type

Events are organized by the top-level `Event Source` key containing an array of `Event Types`

::callout{icon=i-mdi-lightbulb color=amber}
When using the `.on()` method, you can utilize the shorthand `[source].[type]` syntax:
`client.on('Application.ActionAdded', (data) => { console.log(data) })`{lang=ts}
::
