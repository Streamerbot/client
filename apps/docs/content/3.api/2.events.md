## Subscription

::warning
When using the `.on()` method, you can utilize the shorthand `[source].[type]` syntax:
`client.on('Application.ActionAdded', (data) => { console.log(data) })`{lang=ts}
::

::read-more{:to=https://docs.streamer.bot/api/websocket/events}
Explore all event sources and types that can be subscribed to on the [Steamer.bot Docs](https://docs.streamer.bot/api/websocket/events)
::
